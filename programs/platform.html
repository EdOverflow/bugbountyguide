What platform should I use?

Depending on the type of program you plan on launching, you might need to use a different platform to run the triaging process. The most notable bug bounty platforms in no particular order are:

Platform name Website
HackerOne https://www.hackerone.com/
Bugcrowd https://www.bugcrowd.com/
Synack https://www.synack.com/
Cobalt.io https://cobalt.io/
Bounty Factory https://bountyfactory.io/en/index.html
intigriti https://www.intigriti.com/public/
Zerocopter https://www.zerocopter.com/
HackenProof https://hackenproof.com/
